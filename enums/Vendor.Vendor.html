Enumeration Vendor

Enum describing possible vendors we might want to apply prefixes to either CSS properties or values for.

Vendors values are ascending powers of two, allowing encoding information about which vendors apply in a single number. Helper values have been defined on the enum including all possible permutations to save you having to do the bitwise operations yourself.

Example

let numToTest = Vendor.none;

const isNumberAffectedByVendor = (num: number, vendor: Vendor) => {
  return num & vendor;
};

// false
isNumberAffectedByVendor(numToTest, Vendor.moz);

numToTest |= Vendor.mz;

// true
isNumberAffectedByVendor(numToTest, Vendor.moz);
